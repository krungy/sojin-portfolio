# Sojin Portfolio Boilerplate

Next.js(App Router) 기반 포트폴리오 보일러플레이트입니다.

## Stack

- Next.js
- React / React DOM
- TypeScript
- SCSS Modules
- classnames
- framer-motion
- ESLint
- Prettier

## Run

```bash
pnpm dev
```

## Scripts

```bash
pnpm lint
pnpm lint:fix
pnpm format
pnpm format:write
pnpm build
```
